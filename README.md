# l2tp-client
script for l2tp (only pskshared secret) client tested on debian 8

pre-installation:
apt-get install strongswan
apt-get install xl2tpd

================================================
user and password are kept in config.ini 
================================================

How to make a connection:
sudo python3 connect.py
